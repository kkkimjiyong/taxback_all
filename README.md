## 코드 컨벤션
    
- 타입스크립트
    - interface가 아닌 type으로 선언
    - type 앞 변수명 맨 앞글자는 " 대문자 T + type이 사용될 변수명 "
    <br/>
- 폴더 및 파일이름 & 함수는 파스칼케이스
- 변수는 카멜케이스
